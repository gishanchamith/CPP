⚙️ 3. Algorithms

📦 Searching & Sorting

  • binary_search() - Sorted data, O(log n)
  • lower_bound(), upper_bound() - Positions in sorted containers
  • sort() - Efficient built-in sorting
  • Custom sort with lambda/comparator

📐 Math & Number Theory

  • Prime Sieve, GCD/LCM (__gcd)
  • Modular arithmetic, Fast exponentiation
  • Bit Manipulation, Combinatorics (nCr, Pascal’s Triangle)

🔁 Recursion & Backtracking

  • DFS recursion, Backtracking (N-Queens, permutations)
  • Memoization (top-down DP)

🧩 Dynamic Programming (DP)

  • 1D/2D DP (e.g., Fibonacci, Knapsack)
  • State compression (bitmask DP), DP on trees
  • Tabulation (bottom-up), memoization (top-down)

🌐 Graph Algorithms

  • BFS/DFS, Dijkstra, Bellman-Ford, Floyd-Warshall
  • Union-Find (DSU), Topological Sort, Tarjan’s Algorithm, Bridges

🔲 Geometry

  • Point/Line distance, Orientation, Convex Hull, Area calculations

🛠️ 4. STL & Macros

  • vector, pair, map, set, sort(), find(), count(), accumulate()
  • priority_queue, deque, bitset, stack, queue
  • Macros: #define int long long, #define pb push_back, #define all(x) x.begin(), x.end()
  • #define fastio ios::sync_with_stdio(false); cin.tie(0);
