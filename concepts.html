1. Core Concepts

ConceptDescription
VariableA named storage for data. int age = 21;
Data TypesDefines the type of variable (int, char, float, double, bool)
OperatorsSymbols that perform operations: +, -, *, /, %, ==, !=, &&
Control FlowStructures: if, else, switch, for, while, do-while, break, continue
FunctionsReusable code blocks. int add(int a, int b) { return a + b; }
ScopeVariable visibility: local, global, static, extern
Type CastingConvert types: int x = (int)3.14;

2. Object-Oriented Concepts

ConceptDescription
ClassBlueprint: class Car { ... };
ObjectInstance: Car myCar;
EncapsulationGroup data & methods, use of private/public/protected
InheritanceBase class: class B : public A {};
PolymorphismOverloading & overriding
AbstractionInterfaces and abstract classes
Constructor/DestructorInitialize/clean-up an object

4. Advanced Features

FeatureDescription
TemplatesGeneric programming: template<typename T>

NamespacesOrganize code: namespace mySpace { ... }
Exception Handlingtry, catch, throw
File Handling<fstream>: ifstream, ofstream
